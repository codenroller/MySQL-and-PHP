Using MySQL and PHP

The goal of the project is to go through MySQL database basics. We will be focusing on tasks and problems that are common in web development. The main focus of the project is on MySQL database. However it is also important to understand how to combine web technologies to achieve the desired result. So we will go through a couple of examples that use MySQL and PHP together.

Keywords:

MySQL, SQL, PHP, relational database, data integrity, data constraints, data type, table, query, view, transaction.

Assignment:

  1. Create a new database named jeux-video. For starters you have a MySQL code that creates the tables in DB. The initial script is saved in the project root directory as initial_script.sql. Go through the code, apply changes if necessary. Explain the changes you have applied. Execute the final version of your script. Now nou are good to go! :)
  2. Write a query showing all video games in alphabetical order. The query should return: (a) game id; (b) game name; (c) name of owner; (d) name of console; (e) game price. Save your script as qr_video_games.sql. Create a view based on the written query. Name the view vw_video_games and save the script as vw_video_games.sql.
  3. Write a query that displays all the games owned by Patrick. Games should be displayed in alphabetical order. Save the script under the name qr_games_owner_patrick.sql. Write a PHP script displaying a list of video games belonging to each owner. The owners and the games should be listed in alphabetical order. Save the script under the name owners.php.
  4. Write a PHP script displaying a list of video games for each of the consoles. The consoles and the games should be listed in alphabetical order. Save your script under the name consoles.php.
  5. Write a query showing only the games starting from 'M'. The query should return: (a) game id; (b) game name; (c) name of owner; (d) name of console; (e) game price. Name the script qr_video_games_m.sql. To keep things simple you are allowed to use the view vw_video_games that you created in the previous task.
  6. Write a query that displays games with id of a console that is not registered in the table console.
